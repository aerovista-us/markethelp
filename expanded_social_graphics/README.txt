Folders (relative to this directory):
- square_1080/ — 1080×1080 square PNGs, one per day of the 30-day organic cycle
- Vertical/story PNGs — 1080×1920 (or your template size): keep the same basename as the matching square day where possible (days 1–20 ship with the expanded pack; 21–30 optional when you extend reels)

Marketing command center:
- Open ../index.html in a browser. It loads square art from square_1080/ using the filenames below.

Create the ten Day 21–30 square PNGs (no design software required):
1. Open mini_campaign_squares_exporter.html in Chrome or Edge (internet on for first load).
2. Click “Download ZIP (all PNGs)” and wait for the raster pass to finish.
3. Unzip into this folder as square_1080/ so each *.png sits next to days 1–20 artwork.
4. In ../index.html, set imagePendingExportDays to empty: new Set()

Optional: use “Download this PNG” beside a preview to regenerate a single day.
